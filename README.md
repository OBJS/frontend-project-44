## Setup

```bash
git clone https://github.com/OBJS/frontend-project-44.git
```
```bash
npm i
```
```bash
npm link
```

## Start the game

Calculator:
```bash
brain-calc
```

Even or odd:
```bash
brain-even
```

Greatest common divisor:
```bash
brain-gcd
```

Prime or not:
```bash
brain-prime
```

Complement the progression:
```bash
brain-progression
```
