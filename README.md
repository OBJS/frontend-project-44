## Setup

```bash
install
```

## Start the game

Calculator:
```bash
brain-calc
```

Even or odd:
```bash
brain-even
```

Greatest common divisor:
```bash
brain-gcd
```

Prime or not:
```bash
brain-prime
```

Complement the progression:
```bash
brain-progression
```
